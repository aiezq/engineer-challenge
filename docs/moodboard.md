# Moodboard

Локальный substitute для Pinterest/moodboard-части submission.

## Визуальное направление
- Чистый светлый dashboard без “crypto casino” эстетики
- Спокойные холодные оттенки синего как основной trust-signal
- Мягкие границы, спокойные карточки, интерфейс как рабочий инструмент, а не промо-лендинг
- Плотная data-oriented компоновка без декоративной перегрузки

## Что поддерживает продуктовую идею
- B2B/SaaS ощущение вместо “consumer flashy app”
- Фокус на читабельность таблиц и статусов
- Нейтральный визуальный язык, не спорящий с auth-flow

## Ключевые референсы по качествам
- “операционный интерфейс”
- “спокойный финансовый продукт”
- “минимум лишнего шума”
